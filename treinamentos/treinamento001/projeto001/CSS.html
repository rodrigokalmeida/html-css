O que é CSS?

CSS - Cascading Style Sheets O CSS (Cascading Style Sheets) é a linguagem usada para estilizar e definir a aparência visual de uma página HTML. Com o CSS, você pode mudar o layout, as cores, as fontes, o espaçamento, as bordas e muitos outros aspectos visuais dos elementos HTML. O CSS "cascata" significa que as regras de estilo podem se sobrepor, dependendo de sua prioridade ou especificidade. Você pode usar o CSS para tornar seu site mais bonito e responsivo, ou seja, adaptável a diferentes dispositivos.
